The outcomes for a [[Classification]] model can be summarized by the **confusion matrix**

|                            | **Actually Positive(1)**           | **Actually Negative(0)**         |
| -------------------------- | ---------------------------------- | -------------------------------- |
| **Predicted Positive (1)** | True Positives (TPs)<br>*Correct*  | False Positives (FPs)<br>*Error* |
| **Predicted Negative (0)** | False Negatives (FPs)<br>*Correct* | True Negatives (TPs)<br>*Error*  |
For more on different metrics that can be used to evaluate a classification model (like accuracy, recall, precision, F1 score...) see [[Classification#Measuring Classification Quality]]
