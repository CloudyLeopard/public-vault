**Clock**: a small circuity in a digital computer system that generates electrical signal, which goes "up", then "down", then "up", and so on
- **Cycle**: The time between the first "up", then "down", then the second "up"
- **Frequency**: number of cycles in a set time. The higher the frequency, the higher the number of cycles, which means more work done per second. This is measured in **Hertz**, such as *Gigahertz* (1 billion number of cycles per second)
